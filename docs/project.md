# Archivio Centrale dei Custodi del Doge

Versione attuale: 0.1

## Obiettivo

Realizzare il Sistema Informativo Storico della Confraternita dei Custodi del Doge.

WordPress viene usato come motore applicativo, ma l’architettura dovrà permettere una futura migrazione verso una piattaforma autonoma.

## Concetto centrale

Ogni elemento dell’Archivio è un Nodo.

Tipi di Nodo:

- Dossier
- Luogo
- Villa
- Personaggio
- Documento
- Reperto
- Bibliografia
- Evento
- Immagine
- Timeline
- Confratello

Ogni Nodo potrà essere collegato ad altri Nodi.
