# Changelog

## v0.1.0-alpha

Prima milestone pubblica di Xukera Core.

### Aggiunto

- Core separato da Custodi Archivio
- Composer con autoload PSR-4
- PHPUnit
- Test automatici per Node, Relation, Graph, Query e Validator
- Node
- Relation
- Graph
- Query
- Filtri per tipo
- Filtri per metadati
- Traversal del grafo
- incoming()
- outgoing()
- neighbors()
- findPath()
- Validator base
- Documentazione tecnica in docs/
- Documento di architettura Xukera 1.0

### Rimosso

- File legacy applicativi
- Componenti WordPress dal repository Core
- Materiale specifico dei Custodi del Doge

### Note

Xukera è ora un framework PHP indipendente per la gestione della conoscenza.
Custodi Archivio diventa un progetto separato costruito sopra Xukera.
