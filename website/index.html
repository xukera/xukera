Xukera

Universal Knowledge Framework for PHP

Cos'è Xukera

Xukera è un framework PHP per rappresentare, interrogare e salvare conoscenza sotto forma di nodi, relazioni, metadati e grafi.

Non è un CMS. Non è un plugin WordPress. Non è un'applicazione verticale. È un motore universale della conoscenza.

Il cuore del progetto

Xukera permette di lavorare con:

  • Node: entità della conoscenza
  • Relation: collegamenti tra entità
  • Graph: struttura complessiva della conoscenza
  • Query: interrogazione del grafo
  • Storage: salvataggio e caricamento dei dati
  • Serialization: esportazione in formati leggibili
  • Validation: controllo della coerenza del grafo

Esempio

use Xukera\Core\Graph;
use Xukera\Core\Node;
use Xukera\Core\Query;

$graph = new Graph();

$graph->addNode(new Node('villa_widmann', 'luogo', 'Villa Widmann', [
    'comune' => 'Mira',
]));

$results = Query::create($graph)
    ->whereType('luogo')
    ->whereMetadata('comune', 'Mira')
    ->get();

Filosofia

Xukera nasce per essere indipendente da ogni dominio applicativo. Le applicazioni costruiscono significato sopra il motore, senza contaminare il Core.

L'Archivio dei Custodi del Doge sarà la prima applicazione costruita sopra Xukera.
